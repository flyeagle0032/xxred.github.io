## 切换到root

|ubuntu|centos||
|:--|:--|:--|
|sudo su|||

## 删除

- 此命令慎用
- `-rf`后接文件或文件夹

```bash
sudo rm -rf /opt/dotnet/sdk/1.0.4
```

## 软链接

### 创建

```bash
sudo ln -s /src /src-ln
```

### 删除软链接

- 注意，`/src-ln/`多加个斜杠会把`/src`里面的内容也干掉

```bash
sudo rm -rf /src-ln
```

## 查看cpu、进程

```bash
top
```

## 查看内存

```bash
free -h
```
