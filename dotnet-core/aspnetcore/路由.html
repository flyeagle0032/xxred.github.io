# Asp. Net Core 路由处理过程

# 流程

  • 从请求中得到路径，再根据路由模板构造一个完整路由
  • 完整的路由起码包括控制器和方法，所以上一步只要是得到控制器和方法，比如请求的是/，那就用默认值补充完整
  • 接着遍历所有的方法，跟路由进行匹配
  • 方法来源于程序集所有控制器方法
